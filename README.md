## 추가 기능 소개
1. 우선순위 지정 기능
2. 고정 리스트를 표로 출력하는 버튼 생성

### 기능 1
- 리스트 추가 시, 3단계로 (높음, 중간, 낮음) 우선순위를 지정할 수 있습니다.
- 단계별로 함께 리스트들이 정렬됩니다
- 단계별로 리스트의 배경 색깔이 통일됩니다.
- 높음 : 빨강
- 중간: 주항
- 낮음 : 연두

### 기능 2
- 미리 요일별로 고정 리스트를 설정해 놓은 것을 표로 출력해 주는 버튼입니다
- 버튼 클릭 시 표가 출력됩니다
- 닫기 버튼 클릭시 표가 삭제됩니다.
